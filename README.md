Python Snake: Power-Up Edition

A customizable, feature-rich version of the classic Snake game built with Python and Pygame.

Features

Power-Up Apples: Three types of apples with different spawn rarities: Red (+1 length), Orange (+3), and Purple (+5).

Adjustable Grid: Play on a standard 32x32 board or a massive 64x64 board.

Wall Mechanics: Choose between "Solid" walls (crash to lose) or "Wrap" walls (pass through to the other side).

Save System: The game automatically remembers your menu settings for your next session.

Smooth Input: Features an input queue system to prevent missed keystrokes during fast turns.

Controls

Movement: W A S D or Arrow Keys

Menu Navigation: Up/Down to select, Left/Right to adjust

Restart: R (on Game Over)

Return to Menu: M (on Game Over)

Installation and Execution

Ensure you have Python installed.

Install the required Pygame library:

pygames
